# 🌍 WanderWise Travel Agency Website

A modern, responsive travel agency website built using **HTML, CSS, and JavaScript**.  
It allows users to explore destinations, view travel packages, calculate booking prices, and submit travel bookings with validation.

---

## 🚀 Features

- 🏝️ Explore popular travel destinations
- 💼 View and book travel packages
- 💰 Dynamic price calculation based on guests
- 📅 Travel date validation (no past dates allowed)
- 📋 Form validation for secure booking
- 🧾 Booking confirmation with Booking ID
- 📞 Contact form with validation
- 📱 Fully responsive design for all devices
- 🎨 Clean, modern UI with smooth hover effects

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
